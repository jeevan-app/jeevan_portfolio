Human walk_run by the deep learning

The ability to walk and run involves a complex coordination of biomechanics, neural control, and energy use. Walking, often called a controlled fall, requires balancing stability and mobility through alternating swing and stance phases, where the leg moves forward and then supports the body. This coordination enables efficient, smooth movement.

Project Overview

Human Walk/Run Classification Project: Developed a machine learning model using a Convolutional Neural Network (CNN) to classify human walking and running patterns. Integrated the model into a Flask web application for real-time predictions, allowing users to upload input data and receive classification results with image and text display.
