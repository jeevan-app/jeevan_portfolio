Weather App

The Weather App is a responsive and interactive web application that provides real-time weather information. Users can check the weather for any city or their current location, with data such as temperature, humidity, wind speed, and weather conditions.

Project Overview

The Weather App is a web application that provides real-time weather information based on the user's location or city. Built with React.js, it fetches live data from a weather API and displays key details like temperature, humidity, and wind speed. The app is fully responsive, allowing seamless access on all devices, and includes features like city search, geolocation, and unit conversion (Celsius/Fahrenheit). This project highlights my skills in API integration, dynamic content management, and responsive design.
